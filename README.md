# World Rock - Portal de História do Rock

Projeto acadêmico desenvolvido como parte da disciplina de Tecnologias para a Internet na **Uniube**. O objetivo deste portal é fornecer uma imersão na história, gêneros e notícias do mundo do rock, utilizando tecnologias web fundamentais.

## 👨‍💻 Sobre o Autor
- **Nome:** Lucas Daniel de Morais


## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica de todo o conteúdo.
- **CSS3:** Estilização personalizada, incluindo layouts flexíveis, design responsivo e efeitos visuais como bordas arredondadas e sombras de texto.

## 🎸 Funcionalidades do Portal
- **Menu de Navegação:** Acesso intuitivo às seções de discografia, videoclips e contatos.
- **Linha do Tempo:** Conteúdo detalhado sobre a evolução do Rock desde os anos 50 até os anos 2000.
- **Seção de Notícias:** Curadoria de notícias relevantes do cenário musical com links externos.
- **Ranking Interativo:** Tabela técnica classificada por subgêneros (Rock, Heavy Metal, Nu Metal, Black Metal).
- **Mídia:** Incorporação de vídeos (YouTube) para enriquecimento da experiência do usuário.

## 🛠️ Estrutura do Projeto
O projeto está organizado para facilitar a leitura e a manutenção:
- `index.html`: Página principal.
- `styles.css`: Folha de estilo centralizada com todas as definições visuais.
- `/imagens/`: Diretório contendo todos os assets visuais do site.

## 💡 Como executar
1. Clone este repositório para sua máquina local.
2. Certifique-se de manter a pasta `imagens/` no mesmo diretório do arquivo `index.html`.
3. Abra o arquivo `index.html` em qualquer navegador web moderno.
